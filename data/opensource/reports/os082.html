项目概述

Allen AI 2026-05-05 同步 arXiv 论文（2605.02881）发布的最新一代开放 Vision-Language-Action 模型。基于 Molmo2-ER 具身推理 backbone，引入 Action Reasoning Token（执行动作前先生成 100-token 显式推理 trace），覆盖 7 项 sim+real 基准全面超越 Pi-05（平均 +4.7%，长程 +9.3%）。完整开放权重、训练代码与 13M token Action Reasoning 数据集，4×A100 即可训练，是 Pi-05 算力门槛的 1/3。是迄今任何开源 VLA 中最大规模实证研究。

核心特性

  • Molmo2-ER 共享 backbone：与 Allen AI 视觉-语言模型完全共享，一次预训练即可服务通用问答+机器人控制+导航
  • Action Reasoning Token：执行动作前生成 ~100 token 显式推理 trace，从根本上把 VLA 从黑盒转化为"会解释"的可审计控制器
  • 完整开放：模型权重 + 训练代码 + 13M token 推理数据集 + Docker 一键部署，4×A100 GPU 即可训练（Pi-05 需 12×A100）
  • 7 项基准全面 SOTA：LIBERO-Long / SimplerEnv-Bridge / SimplerEnv-Google / CALVIN-D-D / Bridge-V2 Franka / Allegro-Hand / UR5 真机
  • 推理动作频率 30-50 Hz，可在 2×A100 推理；100-token trace 自带人类可审计的失败诊断能力
