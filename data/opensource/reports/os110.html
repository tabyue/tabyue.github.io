项目概述

FunASR 是阿里达摩院开源的工业级语音识别工具包（GitHub 15.2k stars / MIT）。核心：统一 AutoModel 接口 + 工业级预训练模型（Paraformer / SenseVoice / Fun-ASR-Nano）+ Docker 服务部署 + 170× realtime 速度。覆盖 ASR + VAD + 标点 + 说话人分离 + 情感识别全链路。

快速开始

# 安装
pip install funasr

# 一行代码完成 VAD + ASR + 标点 + 说话人
from funasr import AutoModel
model = AutoModel(
    model='iic/SenseVoiceSmall',
    vad_model='fsmn-vad',
    spk_model='cam++',
    device='cuda'
)
result = model.generate(input='meeting.wav')
print(result[0]['text'])
# [00:00.4→00:03.8] Speaker 0: 让我们讨论计划。

# 部署为 API 服务
funasr-server --device cuda  # OpenAI 兼容 API
