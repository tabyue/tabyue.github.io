项目概述

LeRobot 是 Hugging Face 开源的机器人学习框架（GitHub 23.5k stars / Apache-2.0）——统一硬件接口 + 标准化数据集格式 + SOTA 策略算法，让机器人学习研究的门槛降低到「几行代码」。

一句话定位：LeRobot 之于机器人学习，类似 HuggingFace Transformers 之于 NLP。

三大设计原则： 1. 数据标准化：LeRobotDataset 格式（Parquet 存状态/动作 + MP4 存图像）让不同机器人采集的数据可互通 2. 策略即插即用：ACT / Diffusion / SmolVLA 等策略共享统一训练接口 3. 硬件抽象：Robot 基类抽象硬件差异，新硬件只需实现 connect() / teleop() / record() 三个方法

快速开始

# 安装
pip install lerobot

# 可视化已有数据集
python -m lerobot.scripts.visualize_dataset \
    --repo-id lerobot/pusht

# 训练 ACT 策略
python -m lerobot.scripts.train \
    --policy.type=act \
    --env.type=pusht \
    --training.num_epochs=100

# 在 SO100 真机上采集数据
python -m lerobot.scripts.control_robot \
    --robot.type=so100 \
    --control.type=record \
    --repo-id my-dataset
