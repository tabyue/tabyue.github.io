项目概述

Gymnasium 是 Farama Foundation 维护的RL 环境标准 API（GitHub 11.2k stars / MIT）——OpenAI Gym 的官方继承者。所有主流 RL 框架（SB3 / CleanRL / RLinf / TorchRL）都以 Gymnasium 为环境接口标准。

一句话：如果你做 RL，Gymnasium 是你必须知道的第一个库——它定义了 env.reset() / env.step(action) 这套全行业共识 API。

快速开始

import gymnasium as gym

# 创建环境
env = gym.make('CartPole-v1', render_mode='human')

# 交互循环
obs, info = env.reset()
for _ in range(1000):
    action = env.action_space.sample()  # 随机动作
    obs, reward, terminated, truncated, info = env.step(action)
    if terminated or truncated:
        obs, info = env.reset()

env.close()

# 自定义环境
class MyRobotEnv(gym.Env):
    def __init__(self):
        self.observation_space = gym.spaces.Box(low=-1, high=1, shape=(7,))
        self.action_space = gym.spaces.Box(low=-1, high=1, shape=(7,))

    def reset(self, seed=None, options=None):
        return self._get_obs(), {}

    def step(self, action):
        # 执行动作，返回 (obs, reward, terminated, truncated, info)
        return self._get_obs(), 0.0, False, False, {}
