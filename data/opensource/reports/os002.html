项目概述

MuJoCo (Multi-Joint dynamics with Contact) 是 DeepMind 开源的高精度物理仿真引擎（GitHub 11.9k stars / Apache-2.0）。2021 年被 DeepMind 收购后从商业软件转为完全开源。是机器人学 + RL 研究中使用最广泛的物理引擎——dm_control / Gymnasium / robosuite / ManiSkill / IsaacLab 等几乎所有框架底层都用 MuJoCo。

核心优势：精确的接触动力学 + 极快的仿真速度 + 稳定的数值积分 + MJCF 模型描述语言。

快速开始

# 安装
pip install mujoco

# Python 基础使用
import mujoco
import numpy as np

# 加载模型
model = mujoco.MjModel.from_xml_path('humanoid.xml')
data = mujoco.MjData(model)

# 仿真循环
for _ in range(1000):
    mujoco.mj_step(model, data)
    print(f'time={data.time:.3f}, qpos={data.qpos[:3]}')

# 渲染
with mujoco.Renderer(model) as renderer:
    renderer.update_scene(data)
    img = renderer.render()
