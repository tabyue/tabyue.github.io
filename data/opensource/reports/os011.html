项目概述

ORB-SLAM3 是最完整的视觉 SLAM 系统（GitHub 6.5k+ stars / GPLv3）。支持单目 / 双目 / RGB-D + IMU 融合，提供定位 + 建图 + 回环检测 + 地图复用全套能力。是视觉 SLAM 领域的「教科书级实现」。

具身关联：室内移动机器人的视觉定位首选方案（当不使用 LiDAR 时）。

快速开始

# 依赖
sudo apt install libopencv-dev libeigen3-dev

# 编译
git clone https://github.com/UZ-SLAMLab/ORB_SLAM3.git
cd ORB_SLAM3
chmod +x build.sh && ./build.sh

# 跑 TUM 数据集（RGB-D）
./Examples/RGB-D/rgbd_tum \
    Vocabulary/ORBvoc.txt \
    Examples/RGB-D/TUM1.yaml \
    /path/to/TUM/rgbd_dataset_freiburg1_desk \
    /path/to/TUM/associations.txt
