项目概述

NVIDIA Isaac Lab（前身 Isaac Gym / Orbit）是 NVIDIA 官方的GPU 加速机器人学习平台——在 Isaac Sim 之上提供标准化 RL / IL 任务环境。数千环境 GPU 并行 + NVIDIA 全生态（Omniverse / USD / PhysX 5 / RTX 渲染）。是工业界训练 locomotion / manipulation 策略的首选。

快速开始

# 安装（需要 Isaac Sim 4.5+）
git clone https://github.com/isaac-sim/IsaacLab.git
cd IsaacLab
./isaaclab.sh --install

# 训练 Humanoid locomotion
./isaaclab.sh -p scripts/rsl_rl/train.py \
    --task Isaac-Velocity-Rough-Unitree-Go2-v0 \
    --num_envs 4096

# 评估
./isaaclab.sh -p scripts/rsl_rl/play.py \
    --task Isaac-Velocity-Rough-Unitree-Go2-v0 \
    --num_envs 32
