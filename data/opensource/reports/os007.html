项目概述

Legged Gym 是 ETH Zürich RSL 开源的四足/双足机器人运动控制 RL 训练框架（GitHub 3k+ stars）。基于 Isaac Gym GPU 并行仿真 + PPO 算法。ANYmal / Unitree Go1 / Go2 / H1 等四足和人形机器人的 locomotion 策略几乎都是用 Legged Gym 或其变体训练的。

定位：locomotion RL 的事实标准训练框架。

快速开始

# 依赖：Isaac Gym (NVIDIA)
pip install isaacgym  # 需要从 NVIDIA 官网下载

git clone https://github.com/leggedrobotics/legged_gym.git
cd legged_gym
pip install -e .

# 训练 Unitree Go2 locomotion
python legged_gym/scripts/train.py --task=go2

# 可视化
python legged_gym/scripts/play.py --task=go2

Sim-to-Real 关键：域随机化（摩擦 / 质量 / 推力扰动）+ 特权信息教师-学生蒸馏。
