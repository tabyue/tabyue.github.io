项目概述

Segment Anything (SAM) 是 Meta AI 发布的通用图像分割基础模型（GitHub 48k+ stars）。给定任意提示（点/框/文本），SAM 能分割图像中的任意物体——零样本、无需针对特定类别训练。

具身关联：机器人抓取前需要精确分割目标物体 → SAM 提供即插即用的通用分割能力，替代过去为每个物体训练专用分割网络的做法。

系统架构

┌─────────────────────────────────────┐
│          SAM 架构                    │
├─────────────────────────────────────┤
│  Image Encoder (ViT-H / ViT-L)     │
│  → 对整张图像只编码一次              │
├─────────────────────────────────────┤
│  Prompt Encoder                     │
│  → 点 / 框 / 文本 / 掩码 提示编码    │
├─────────────────────────────────────┤
│  Mask Decoder (轻量 Transformer)    │
│  → 输出多个候选掩码 + 置信度分数      │
└─────────────────────────────────────┘

关键设计：图像只编码一次（重复使用），每次新提示只过轻量 decoder → 实时交互式分割。

快速开始

from segment_anything import sam_model_registry, SamPredictor
import cv2

sam = sam_model_registry['vit_h'](checkpoint='sam_vit_h.pth')
predictor = SamPredictor(sam)

image = cv2.imread('scene.jpg')
predictor.set_image(image)

# 点击一个点 → 分割该物体
masks, scores, _ = predictor.predict(
    point_coords=np.array([[500, 300]]),
    point_labels=np.array([1]),
)
print(f'Best mask score: {scores[0]:.3f}')
