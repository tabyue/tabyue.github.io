项目概述

Stable Baselines3 (SB3) 是PyTorch 版本的强化学习算法标准库（GitHub 9k+ stars / MIT）。提供 PPO / SAC / TD3 / A2C / DQN / HER 等经典 RL 算法的可靠实现——重点是「可靠」：每个算法都经过严格测试和基准验证。

定位：SB3 之于 RL 算法，类似 scikit-learn 之于传统 ML——不是最快的，但是最「靠谱 + 易用」的。

快速开始

from stable_baselines3 import PPO
import gymnasium as gym

# 5 行代码训练一个 RL Agent
env = gym.make('CartPole-v1')
model = PPO('MlpPolicy', env, verbose=1)
model.learn(total_timesteps=100000)

# 评估
obs, _ = env.reset()
for _ in range(1000):
    action, _ = model.predict(obs, deterministic=True)
    obs, reward, done, _, _ = env.step(action)
    if done:
        obs, _ = env.reset()
