项目概述

ALOHA / Mobile ALOHA 是 Stanford 开源的低成本双臂遥操作 + 学习系统（GitHub 4k+ stars）。ALOHA = A Low-cost Open-source HArdware。$20K 以内搭建完整双臂遥操作平台 + ACT 策略学习。Mobile ALOHA 加了移动底盘，实现全身移动操作。

为什么核心：ALOHA 是当前学术界最广泛使用的双臂数据采集平台——大量 VLA 论文的训练数据来自 ALOHA 系统。

快速开始

# 硬件购买清单（~$20K）
# - 4x Interbotix ViperX 300 6DOF ($3,500 each)
# - 4x Logitech C922 USB Camera ($80 each)
# - 1x Workstation with GPU
# - Mobile ALOHA: + AgileX Tracer底盘 ($5K)

# 软件
git clone https://github.com/tonyzhaozh/aloha.git
cd aloha
pip install -e .

# 采集数据（50 条演示）
python record_episodes.py --task_name my_task --num_episodes 50

# 训练 ACT 策略
python train.py --task_name my_task --policy_class ACT \
    --chunk_size 20 --batch_size 8

# 评估
python eval.py --task_name my_task --ckpt_dir outputs/
