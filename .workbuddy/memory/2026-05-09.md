## 每日英文 v3.1：音标查询（10:48）
- 用户问"能否补音标"——直接 JSON 写死有问题（缩写无标准 IPA / 多义词上下文不同 / 维护成本高）
- 改用**点击实时查询**方案：
  - vocab 词点击 → fetch dictionaryapi.dev 拿 IPA + 真人发音 mp3（免费 API，CORS 允许）
  - 弹出 ipa-pop popover：单词 + IPA + 🔊 播放按钮 + 词义 + 中文释义 + 跳转剑桥/有道/韦氏链接
  - mark.de-vocab（英文段落里的高亮词）+ vocab-list 中的 b 元素都支持点击
  - 查询结果缓存 _ipaCache，同词不重复请求
  - API 找不到（缩写/词组）时 fallback 到外链（剑桥/有道/Wiktionary）
  - 点击 popover 外面或 ✕ 按钮关闭
- CSS：.ipa-pop 浮层 + .ipa-phon（IPA 字体 'Doulos SIL' / 'Charis SIL' fallback Georgia）+ .ipa-play 圆形按钮 + .ipa-links 外链 chips
