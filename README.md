# 具身智能知识门户 | Embodied AI Knowledge Hub

一站式具身智能从业与研究人员知识门户。

**🔗 在线访问：https://tabyue.github.io/**

## 📊 内容板块

- 🗺️ **学习路径** - 从数学基础到项目实战的完整 Step by Step 学习路线图
- 📄 **论文解读** - 精选领域核心论文，含 TL;DR、关键洞察、影响力评估
- 💻 **开源项目** - LeRobot、MuJoCo、Isaac Lab 等核心项目详解
- 📰 **行业新闻** - 产业动态、融资创投、技术突破等每日更新
- 🔗 **资源导航** - 企业图谱、顶级会议、学习社区

## 🤖 自动更新

本站内容由 AI 自动化驱动，每天 00:00 和 12:00 自动更新：
- 搜索最新行业新闻和论文
- 更新开源项目信息
- 补充学习路径资源
- 所有历史内容持续沉淀累积

## 🏗️ 技术架构

- 纯静态 HTML + JSON 数据驱动
- GitHub Pages 自动部署
- 数据文件独立存储，页面自动渲染

## 📁 目录结构

```
├── index.html          # 主页面（SPA）
├── data/
│   ├── learning-path.json   # 学习路径数据
│   ├── papers.json          # 论文解读数据
│   ├── opensource.json       # 开源项目数据
│   └── news.json            # 新闻与资源数据
└── README.md
```
